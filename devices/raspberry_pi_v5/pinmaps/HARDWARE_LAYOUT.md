# Raspberry Pi 5 — Hardware Layout

**Board:** Raspberry Pi 5  
**SoC:** BCM2712 + RP1 (GPIO/peripheral controller)

---

## 40-Pin GPIO Header

Pinout is **identical** to Pi 4 and Pi Zero 2 W: same 3V3, 5V, I2C (3/5), UART (8/10), SPI, PWM, GND, and GPIO0–27. GPIO managed by **RP1**; see [RP1 peripherals](https://www.raspberrypi.com/documentation/microcontrollers/rp1.html). All 3.3 V.
