# Rock64 — Hardware Layout

**Board:** PINE64 Rock64  
**SoC:** Rockchip RK3328

---

## 40-Pin GPIO Header

Raspberry Pi–compatible 40-pin layout: 3V3, 5V, I2C (3/5), UART (8/10), SPI, PWM, GND, GPIO. 3.3 V logic. See [PINE64 Rock64 wiki](https://wiki.pine64.org/wiki/ROCK64) for exact RK3328 GPIO mapping.
